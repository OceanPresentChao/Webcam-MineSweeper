# Webcam-MineSweeper

<p align='center'>
<a href='./README.EN.md'>English</a> | 简体中文
</p>

将扫雷和webcam相结合。既可以通过鼠标来游玩扫雷游戏；也可以通过捕捉摄像头照片来训练模型，让模型识别图像结果来执行结果对应的操作。

## 如何游玩
1. 允许浏览器使用摄像头
2. 摆出不同的动作，点击按钮让程序捕捉图像
3. 对每种操作都采集足够多且不同的样本后，点击训练
4. 点击预测。模型会预测当前的动作执行相应的操作
